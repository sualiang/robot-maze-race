## 三、其他端口测试结果

### 3.1 运营商端
- **登录方式**: 手机号 + 密码
- **状态**: ❌ 登录失败
- **详情**: 总后台"重置密码"生成随机密码（含`%&^`等特殊字符），3个运营商均无法登录
- **可能原因**: 登录接口对特殊字符未做URL编码处理；或运营商需激活

### 3.2 商家端
- **状态**: ❌ 未测试（未获得有效商家账号）
- **问题**: 总后台运营商列表中的"商家"按钮点击后无页面跳转，商家管理入口不明确

### 3.3 裁判端 🔴 致命Bug
- **登录方式**: 微信服务号网页授权（OAuth）
- **登录地址**: https://amberrobot.com.cn/referee/login
- **错误**: 点击"微信授权登录"后跳转到 `http://open.weixin.qq.com/...?redirect_uri=http%3A%2F%2Famberrobot.com.cn%2Freferee%2Flogin`
- **根因**: 后端代码中`redirect_uri`写死为`http://amberrobot.com.cn`，但微信开放平台安全域名必须为HTTPS
- **现象**: 微信报错"redirect_uri参数错误"，即使在手机微信内置浏览器中同样失败
- **严重程度**: 🔴 裁判端完全不可用

### 3.4 大屏端
- **状态**: ⏭️ 跳过（用户确认功能简单，无需测试）
- **登录方式**: 裁判小程序扫码激活Canvas动态二维码（60秒刷新）
