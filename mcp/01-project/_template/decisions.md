## 📋 决策记录格式

每条决策记录必须包含：
- **决策编号：** 唯一标识，如 DEC-001
- **决策时间：** YYYY-MM-DD
- **决策人：** 最终拍板的人（人类/某角色）
- **决策背景：** 为什么要做这个决策
- **可选方案：** 有哪些备选方案
- **决策内容：** 最终决定是什么
- **决策原因：** 为什么选这个方案
- **影响范围：** 影响哪些模块/角色
- **后续动作：** 需要做什么来落实

## ⚠️ 决策变更

如果后续需要变更已有的决策，必须：
1. 说明变更原因
2. 重新走决策流程
3. 在原决策记录上标注"已变更"，并链接到新的决策
4. 通知所有受影响的角色

**禁止**悄悄变更决策而不记录。
